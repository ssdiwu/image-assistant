## English

### ğŸ“– Introduction

Image Assistant is a Claude Code Skill that helps you convert article or module content into unified-style, text-minimal, highly readable 16:9 infographic prompts.

### âœ¨ Use Cases

- Need illustrations for articles but don't know how to design
- PPT, posters, or social media graphics need a unified style
- Too much text, want more engaging and readable visual presentation
- Need to batch generate illustration prompts

### ğŸ¯ Core Features

- ğŸ“‹ **Requirement Clarification**: Extract content, scenario, audience, and text density preferences
- ğŸ—‚ï¸ **Illustration Planning**: Split content, define image list (how many/what each explains)
- âœï¸ **Copy Finalization**: Word-by-word finalization of "what text goes on the image" (Copy Spec)
- ğŸ¯ **Prompt Packaging**: Generate copy-ready image generation prompts, support batch generation
- ğŸ”„ **Iterative Refinement**: Reduce text, change metaphors, improve readability based on feedback
- ğŸ’¬ **Session State Recovery**: Stage progress is tracked so you can resume seamlessly after interruption
